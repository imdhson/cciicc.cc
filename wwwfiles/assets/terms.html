이용 약관

개요

본 서비스는 분리된 공간에서 텍스트 등을 다수의 사용자와 주고받을 수 있는 서비스입니다.

용어와 정의

provider : cciicc.cc를 운영하는 자

user : cciicc.cc를 사용하는 자

host : space를 새로 생성하여 입장하는 user

guest : 이미 만들어진 space에 입장하는 user

comment : 각 space에 존재하는 데이터.(주제, 아이디어, 대화 등)

space: user들이 집합되어 있으며 user가 참여하고 있는 하나의 공간. user와 comment 데이터를 포함함.

쿠키의 이용

본 서비스는 데이터 업로드 시 사용자를 식별하기 위해 웹브라우저의 cookie를 사용합니다. cciicc.cc를 이용하는 것은 cookie 사용에 동의하는 것으로 간주됩니다.

베타 서비스

본 서비스는 충분한 테스트가 이루어지지 않는 베타 서비스 입니다. 중대한 작업을 본 서비스에서 수행하는 것은 권장되지 않으며, 서비스 이용 중 재산상의 손해 등이 발생하여도 provider가 책임지지 않습니다.

이용자 간 분쟁

본 서비스는 서버 측에서 데이터가 지극히 짧은 시간 동안 유지되고, 로그인이 존재하지 않아 이용자 간의 분쟁이 생기더라도 provider가 분쟁을 제지하거나 관련한 데이터를 보존, 제공 할 수 없습니다.

개인정보 보호 정책

개인정보의 생성

host, guest로 space에 입장 할 때 동시에 고유한 식별자가 발급됩니다. 식별자와 연관된 데이터는 user 이름(nickname), host 여부, 참가 중인 space의 id값이 있습니다.

개인정보의 활용

user가 정상적으로 서비스를 이용할 수 있도록 고유 식별자를 이용하여 필요한 데이터를 서버에서 전송하게 됩니다. 해당 목적 이외로 사용되지 않습니다.

space에 저장된 comment에는 user 이름(nickname), 본문 내용, 좋아요 개수, 배경 색상등이 저장되어 활용되게 됩니다. 해당 목적 이외로 사용되지 않습니다.

개인정보의 파기

유휴 space의 파기: space에서 1시간 이상 새로운 comment, 좋아요 전송 등이 일어나지 않으면 해당 space를 삭제하게 됩니다. space의 포함된 모든 데이터가 삭제됩니다. space를 삭제할 때 별도의 백업 절차 없이 실제 데이터를 완전히 서버에서 삭제하게 됨으로 복구가 불가능합니다.

유휴 user의 파기: space의 삭제를 수행 할 때 참가 중인 space의 id값이 같은 user를 삭제하게 됩니다. user의 포함된 모든 데이터가 삭제됩니다. user를 삭제할 때 별도의 백업 절차 없이 실제 데이터를 완전히 서버에서 삭제하게 됨으로 복구가 불가능합니다.

유휴 space, user를 찾는 과정은 서버에서 정기적으로 수행하게 됩니다. 따라서 정확히 1시간이 지난 직후에 삭제되지 않을 수 있습니다. 서버 환경에 따라 최대 1시간 지연될 수 있습니다. 따라서 마지막 상호작용 이후 최대 2시간이 지난 이후 삭제될 수 있습니다.
